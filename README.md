# CS330
In CS 330: Computational Graphics and Visualization, I designed and constructed a fully interactive 3D desk scene with C++ and OpenGL. The project entailed transforming a 2D reference image into low-polygon 3D models made of basic shapes including boxes, cylinders, cones, and planes. I added textures, used the Phong shading model to create various light sources, and included full camera navigation with perspective and orthographic projection options. Throughout development, I used an iterative design process, starting with core geometry and then refining object placement, materials, lighting, and camera settings to get a professional final presentation.

This project helped me improve my knowledge of rendering pipelines, transformations, shader management, and modular programming structure. I took a more deliberate approach to software design, separating functionality into reusable systems and developing them incrementally. The experience helped me improve my spatial reasoning, debugging skills, and ability to balance visual correctness and performance. These computational graphics techniques lay a solid foundation for future work in game development, simulation, visualization, and interactive software systems, while also reinforcing fundamental computer science principles like structured problem solving and iterative development.
